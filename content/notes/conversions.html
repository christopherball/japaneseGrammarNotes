Noun → Adjective

Pattern How it works Examples
Noun + 的 Forms a な-adjective from a subset of pure nouns. This pattern is more common with abstract concepts and has tighter lexical limits than + の. 文化 → 文化的
culture → cultural

歴史 → 歴史的
history → historical

科学 → 科学的
science → scientific
Noun + の Works productively with pure nouns. Grammatically this is an adjectival noun phrase rather than a true adjective, but it behaves like an adjective in practice. 車の本
a book about cars

木の椅子
a wooden chair

Warning: Noun + 的 is not freely available for every noun, so it is safer to learn it as a lexical pattern rather than assuming it can always be formed.

Adjective → Adverb

Pattern How it works Examples
い-adjective stem + く Removes the final い and turns the adjective into an adverbial form. 私は京都に長く住んでいました。
I lived in Kyoto for a long time.

そのラーメンを早く食べました。
I ate that ramen quickly.
な-adjective stem + に Uses the adjective stem and adds に. もっと効率的に日本語が勉強したいです。
I want to study Japanese more efficiently.

図書館に雑誌を静かに読んでいることがよくする。
I often read magazines quietly in the library.

Adjective → Noun

Pattern Quality type Examples
い-adjective stem + さ Objective or measurable quality. 高い → 高さ
tall / expensive → tallness / expensiveness
な-adjective stem + さ Objective or measurable quality. 便利な → 便利さ
convenient → convenience
い-adjective stem + み Subjective, experiential, or felt quality. 面白い → 面白み
interesting → interest / interestingness
な-adjective stem + み Subjective quality, but only for a small set of adjectives. 新鮮な → 新鮮み
fresh → freshness

Warning: + み is much more lexically restricted than + さ, especially with な-adjectives.

Verb → Noun

Pattern How it works Examples
Plain verb + の Nominalizes the verb phrase. In practice it behaves like a noun phrase. 甘い物を食べるのが好きだ。
I like eating sweet things.
Plain verb + こと Another nominalizer, often slightly more formal or abstract than の. 全部の漢字を暗記する事ができないかな。
I wonder if I can memorize all the kanji.
Verb ます-stem Some verbs have lexicalized noun uses in the ます-stem. This is not productive across all verbs. 富士山は年をとるの登り難しくなった。
As I get older, it has become more difficult to climb Mt. Fuji.

今日ここから富士山は見やすいです。
It's easy to see Mt. Fuji from here today.

Warning: Stem-based noun readings are lexical items, not something you can safely generate for every verb.
