This page collects common Japanese word-class conversion patterns: ways nouns become adjective-like, adjectives become adverbs or nouns, and verbs become noun-like expressions.

Noun -> Adjective

Pattern How it works Examples
Noun + 的 -> な-adjective Forms a な-adjective from a subset of pure nouns. This pattern is more common with abstract concepts and has tighter lexical limits than + の. 文化ぶんか -> 文化的ぶんかてき
culture -> cultural

歴史れきし -> 歴史的れきしてき
history -> historical

科学かがく -> 科学的かがくてき
science -> scientific
Noun + の -> adjective phrase Works productively with pure nouns. Grammatically this is an adjectival noun phrase rather than a true adjective, but it behaves like an adjective in practice. 車くるまの本ほん
car book / a book about cars

木きの椅子いす
wooden chair

Warning: Noun + 的 is not freely available for every noun, so it is safer to learn it as a lexical pattern rather than assuming it can always be formed.

Adjective -> Adverb

Pattern How it works Examples
い-adjective stem + く -> adverb Removes the final い and turns the adjective into an adverbial form. 私わたしは京都きょうとに長ながく住すんでいました。
I lived in Kyoto for a long time.

そのラーメンを早はやく食たべました。
(I) quickly ate that ramen.
な-adjective stem + に -> adverb Uses the adjective stem and adds に. もっと効率的こうりつてきに日本語にほんごが勉強べんきょうしたいです。
(I) want to study Japanese more efficiently.

図書館としょかんに雑誌ざっしを静しずかに読よんでいることがよくする。
(I) often quietly read magazines in the library.

Adjective -> Noun

Pattern Quality type Examples
い-adjective stem + さ -> noun Objective or measurable quality. 高たかい -> 高たかさ
tall / expensive -> tallness / expensiveness
な-adjective stem + さ -> noun Objective or measurable quality. 便利べんりな -> 便利べんりさ
convenient -> convenience
い-adjective stem + み -> noun Subjective, experiential, or felt quality. 面白おもしろい -> 面白おもしろみ
interesting -> interest / interestingness
な-adjective stem + み -> noun Subjective quality, but only for a small set of adjectives. 新鮮しんせんな -> 新鮮しんせんみ
fresh -> freshness

Warning: + み is much more lexically restricted than + さ, especially with な-adjectives.

Verb -> Noun

Pattern How it works Examples
Plain verb + の -> noun Nominalizes the verb phrase. In practice it behaves like a noun phrase. 甘あまい物ものを食たべるのが好すきだ。
(I) like to eat sweet things.
Plain verb + こと -> noun Another nominalizer, often slightly more formal or abstract than の. 全部ぜんぶの漢字かんじを暗記あんきする事ことができないかな。
(I) wonder if I can memorize all the kanji.
Verb ます-stem -> noun Some verbs have lexicalized noun uses in the ます-stem. This is not productive across all verbs. 富士山ふじさんは年としをとるの登のぼり難むずかしくなった。
As I get older, it has become more difficult to climb Mt. Fuji.

今日きょうここから富士山ふじさんは見みやすいです。
It's easy to see Mt. Fuji from here today.

Warning: Stem-based noun readings are lexical items, not something you can safely generate for every verb.
